# BLACKSTONE COFFEE

Small multi‑page coffee shop website built with HTML, CSS, and JavaScript (Frontend).

## Pages

- index.html – Home / landing page for the shop.  
- products.html – Drink cards with filter buttons (All, Espresso, Coffee, Other).
- contact.html – Address, phone, email, and hours section.

## Tech

- HTML for structure  
- CSS for layout and styling  
- Vanilla JavaScript for loading drinks and filtering

## How to Run

1. Download the project files.  
2. Open index.html in browser or local server.  

## Customization

- Edit drink data and images in product.js.  
- Update contact info and shop name text in the HTML files.
- Use a different API for coffees (that actually works).
